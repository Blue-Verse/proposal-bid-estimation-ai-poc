### 사전 검증 질문 답변

**Q1. 프로젝트와 관련된 사업 분야 업무 경험에 대해 설명해주세요.**
AI 기반 문서 처리 및 데이터 자동화 분야에서 다수의 프로젝트를 수행했습니다. 특히 비정형 PDF 문서(계약서, 보고서, 품셈집 등)에서 구조화된 데이터를 추출하고, RAG 파이프라인을 통해 의미론적 검색·매칭 시스템을 구축한 실전 경험을 보유하고 있습니다. 건설·설비 도메인의 견적 업무 프로세스에 대한 이해를 바탕으로, 현업 용어와 표준 품셈 용어 간의 매칭 정확도를 높이는 하이브리드 검색(벡터 유사도 + BM25) 접근법을 제안드립니다.

**Q2. 유사한 기술을 사용한 경험에 대해 설명해주세요.**
- **RAG 파이프라인**: LangChain + OpenAI Embeddings + ChromaDB 기반의 문서 인덱싱 및 의미론적 검색 시스템을 다수 구축했습니다.
- **PDF 데이터 추출**: PyMuPDF + pdfplumber를 활용한 7중 PDF 처리 파이프라인(텍스트, 표, 이미지, 메타데이터 등)을 운영 중입니다.
- **대규모 데이터 그리드**: AG Grid 기반 3,000건 이상의 데이터를 실시간 조회·수정·저장하는 웹 UI를 구현한 경험이 있습니다.
- **FastAPI + Next.js**: Python AI 백엔드와 React 프론트엔드를 결합한 풀스택 시스템을 반복적으로 구축해 왔습니다.

**Q3. 유사한 프로젝트를 수행한 경험이 있다면 무엇입니까?**
1. **AI-Native Development Framework** — RAG 기반 문서 검색·매칭 엔진, PDF 구조화, 의미론적 유사도 분석을 핵심으로 하는 AI 에이전트 프레임워크를 자체 개발·운영 중입니다.
2. **Series B (투자조합 관리 플랫폼)** — 수백 건의 비정형 PDF(투자계약서, 재무제표)에서 데이터를 자동 추출하고, 7중 처리 파이프라인과 대규모 그리드 UI를 구현했습니다.
3. **EZ-Approve (전자결재 플랫폼)** — 복잡한 업무 시스템의 웹 UI를 3주 내 단기 집중 개발로 완성한 경험이 있습니다.
